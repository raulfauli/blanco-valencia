## Why

The site lacks a custom 404 error page. When users navigate to non-existent URLs, they see a generic browser error or Astro's default page, which provides a poor user experience and misses an opportunity to keep visitors engaged with the site.

## What Changes

- Create a custom 404 page with branded styling matching the site design
- Provide helpful navigation options to guide users back to valid content
- Display a friendly error message in Spanish

## Capabilities

### New Capabilities

- `error-404-page`: Custom 404 error page with navigation options and branded design

### Modified Capabilities

- None

## Impact

- New file: `src/pages/404.astro`
